演示如何打印条形码

条码语句

ADD_PRINT_BARCODE(Top, Left, Width, Height, BarCodeType, BarCodeValue);

语句参数

参数名 类型 说明
Top 数值 条码的位置Y值，单位px（1px=1/96英寸）
Left 数值 条码的位置X值，单位px
Width 数值 条码的总宽度，单位px
Height 数值 条码的总高度（一维条码时包括文字高度）
BarCodeType 字符串 条码的类型（规制）名称
BarCodeValue 字符串 条码值

条码类型

目前控件支持的条码类型共27种，其中一维码24种，二维码3种：

一维码（24种）
Code 128系列：
128A 128B 128C 128Auto
EAN系列：
EAN8 EAN13 EAN128A (或GS1-128A) EAN128B (或GS1-128B) EAN128C (或GS1-128C)
UPC系列：
UPC_A UPC_E0 UPC_E1 UPCsupp2 UPCsupp5
Code 39系列：
Code39 39Extended Code93 93Extended
2 of 5系列：
2_5interleaved 2_5industrial 2_5matrix
其他一维码：
MSI PostNet Codabar
二维码（3种）
QRCode PDF417 DataMatrix
